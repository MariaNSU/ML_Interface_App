MHAD2012-6

Основная информация

сканирование тэг заходы релиз путь
MHAD2012 MHAD2012-6 11285..13219 R006-004 /online/proc/MHAD2012-6
  • Меморандум о шумах в дрейфовой камере
  • Энергии измеренные по каонам (КМД3)

Особенности

  • Мюонная система:
    • Заходы с отказом модуля: 11915-11941 (TU7-10, 15).
    • Заходы с низкой эффективностью: 11395, 11688, 11693, 11951, 12062 (везде eff=0). В заходах 12373-12378 эффективность для отдельных модулей не измерялась из-за отказов сцинтилляционных счетчиков. То-есть реального снижения эффективности, скорее всего, нет.
    • Заходы с отказами AC: 11395 (AC10), 11693 (AC16), 12062 (AC18), 12211-12220 (AC15), 12275 (AC18), 12373-12378 (AC04).
    • Возможный "звон" трубки 256 (модуль 15, трубка 30) в заходе 13036.

Основные проблемы, которые ещё не исправлены

Связанные тикеты

  • Низкая эффективность дрейфовой камеры (см. тикет #1159 )
  • Шумящие каналы калориметра (см. тикет #1157 )
  • Поиск пиков в распределении по phi (см. тикет #1354 )
  • Удаление заходов (см. тикеты #1348 б, #1259 )
  • Работа мюонной системы (см. тикет #1745 )

Зависимость различных параметров от номера захода

  • Отношение числа событий Nee/Ngg

  • Среднее энерговыделение нормированное на удвоенную энергию пучка для событий bebe36

  • Минимальная эффективность слоя дрейфовой камеры

  • RMS dtheta для событий bebe36

  • RMS dphi для событий bebe36

  • Проверка на однородность распределения по phi для bebe36

  • Количество событий с наложениями

  • Эффективность черенковского счетчика

  • Эффективность внешней системы
