MHAD2010-4

Основная информация

сканирование тэг заходы релиз путь
MHAD2010 MHAD2010-4 5781..7478 R006-004 /online/proc/MHAD2010-4

Особенности

  • Мюонная система:
    • Заходы с несрабатыванием хотя бы одного модуля: 5851 - 5904, 5913 - 5927, 6539, 7196, 7286, 7365, 7373, 7389, 7390, 7401.
    • Низкая эффективность трубок:
      • 5781-5783: TUEff=0 (TU04)
      • 6921-6923: TUEff=0.3-0.75 для всех модулей.

Основные проблемы, которые ещё не исправлены

  • Обновление калибровки дрейфовой камеры x(t), похоже низкая стат. точность калибровки (тикеты #1375 и #1354 )

Связанные тикеты

  • Заходы с эффективностью порогов < 100% (тикет #1364 )
  • Проблемы с калибровками дрейфовой камеры (тикет #1154 )
  • Шумящие каналы калориметра (тикет #1155 )
  • Удаление заходов (тикет #1348 )

Зависимость различных параметров от номера захода

  • Отношение числа событий Nee/Ngg

  • Среднее энерговыделение нормированное на удвоенную энергию пучка для событий bebe36

  • Минимальная эффективность слоя дрейфовой камеры

  • RMS dtheta для событий bebe36

  • RMS dphi для событий bebe36

  • Проверка на однородность распределения по phi для bebe36

  • Количество событий с наложениями

  • Эффективность черенковского счетчика

  • Эффективность внешней системы
