RHO_2019-2 (43302..44653,R008-002)

Основная информация

сканирование тэг заходы релиз путь
RHO_2019 RHO_2019-2 43302..44653 R008-002 /online/proc2/RHO_2019-2

Особенности

  • В заходах 43503-43508 и 44547 подозрительно маленькое отношение Nee/Ngg
  • Заход 44621 был обрезан до 880к событий
  • У заходов с 44240 по 44350 низкая эффективноть 1-го и 4-го слоёв ДК + очень маленькая светимость + низкий процент заходов, которые закончились по числу событий
  • Низкое отношение числа событий cbebe45 к светимости в заходах 43774 и 44575

Зависимость различных параметров от номера захода

  • Энергия пучка в зависимости от номера захода

  • Отклонение Nee/Ngg от ожидаемого значения в единицах стат. ошибок для разных заходов

  • Число событий в gcl-файлах

  • Наименьшая эффективность слоя ДК

  • Наименьшая эффективность АЧС

  • Разброс dphi для cbebe36 в зависимости от номера захода

  • Зависимость разброса dphi для cbebe36 от энергии объясняется многократным рассеянием

  • Ширина на полувысоте распределения dtheta для cbebe36

  • Разброс dphi для cgaga36 в зависимости от номера захода

  • Разброс dtheta для cgaga36 в зависимости от номера захода

  • Среднне значение Eton для cbebe36

  • Время z24 скачет на такт из-за нерешенных проблем с т.н. Жулановским битом, но для определения фазы такой точности достаточно

  • Отношение числа событий cbebe45 к светимости (по FLT1)
