MHAD2011-6

Основная информация

сканирование тэг заходы релиз путь
MHAD2011 MHAD2011-6 7842..10988 R006-004 /online/proc/MHAD2011-6
  • Энергии измеренные по каонам (КМД3)

Особенности

  • Работа трубок мюонной системы:
    • В заходах 7842-7899 трубки мюонной системы, видимо, не работали совсем.
    • В заходах 10503-10555 не работал модуль 10 трубок.
    • В заходах 8571-8603 очень низкие загрузки трубок (в пределах 0.005 Гц для большинства трубок). Работа системы явно не была нормальной. По-видимому, эти заходы не следует использовать.
    • Заходы со сниженной эффективностью мюонных трубок:
      • 10457 - все модули с эффективностью не выше 0.9.
      • 10502 - эффективность TU10 0.86
      • 10889 - все модули с эффективностью не выше 0.9.
      • 10892 - большинство модулей с эффективностью не выше 0.9.
      • 10898 - все модули с эффективностью не выше 0.7.

Основные проблемы, которые ещё не исправлены

Связанные тикеты

  • Низкая эффективность дрейфовой камеры (см. тикет #1158 )
  • Шумящие каналы калориметра (см. тикет #1156 )
  • Поиск пиков в распределении по phi (см. тикет #1354 )
  • Удаление заходов (см. тикеты #1348 , #1258 и #1357 )
  • Отсутствовали калибровки nainai для точек 812.5, 837.5, 862.5, 887.5, 912.5, 935, 945 (см. тикет #958 )

Зависимость различных параметров от номера захода

  • Отношение числа событий Nee/Ngg

  • Среднее энерговыделение нормированное на удвоенную энергию пучка для событий bebe36

  • Минимальная эффективность слоя дрейфовой камеры

  • RMS dtheta для событий bebe36

  • RMS dphi для событий bebe36

  • Проверка на однородность распределения по phi для bebe36

  • Количество событий с наложениями

  • Эффективность черенковского счетчика

  • Эффективность внешней системы
